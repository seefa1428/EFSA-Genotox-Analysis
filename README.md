# EFSA-Genotox-Analysis  
## EFSA Pre-Interview Task 2026  

This repository contains the code and analysis for the EFSA Pre-Interview Task on genotoxicity data of pesticide active substances and their metabolites.

## Contents

- Jupyter Notebook with all code, outputs, and analysis  
- Data preprocessing, exploration, and aggregation steps  
- Q1a–Q1d solutions implemented in Python  

## Methodology

- Data cleaning and standardization  
- Group-by analysis at substance level  
- Filtering based on EFSA criteria (including post-2013 opinion data)  
- Aggregation of assay-level results into structured summaries  
- Generation of final tables and insights  

## Tools Used

- Python  
- Pandas  
- NumPy  
- Jupyter Notebook
